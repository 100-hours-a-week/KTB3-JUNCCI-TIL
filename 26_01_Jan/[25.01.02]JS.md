# 📚 Js - Template Literal, Shallow Comparison, Deep Comparison 정리

---


# <mark> 템플릿 리터럴 (Template Literal) </mark>

### 개념

* 문자열을 더 편하게 만드는 **ES6 문법**
* **백틱(`)** 사용

```js
`문자열`
```

---

### 핵심 기능

#### 변수 / 표현식 삽입

```js
const name = "민수";
const age = 20;

`이름: ${name}, 나이: ${age}`
```

#### 연산 & 함수 사용

```js
`합계: ${10 + 5}`
`대문자: ${name.toUpperCase()}`
```

#### 멀티라인 문자열

```js
`
첫 줄
두 번째 줄
`
```

---

### 핵심 요약

* `${}` 안에는 **표현식만 가능**
* 가독성 ↑, 문자열 연결(`+`) 대체

---

# <mark> 얕은 비교 (Shallow Comparison) </mark>

### 개념

* **값이 아니라 참조(주소)** 를 비교
* 객체, 배열, 함수에 적용됨

```js
const a = { x: 1 };
const b = { x: 1 };

a === b; // false
```

```js
const c = a;
c === a; // true
```

---

### 특징

* 내용이 같아도 **참조가 다르면 false**
* `===`, `==` 모두 객체에서는 얕은 비교

```js
[] === []; // false
```

---

### 사용 예

* React 상태 변경 감지
* 성능이 중요할 때

---

# <mark> 깊은 비교 (Deep Comparison) </mark>

### 개념

* 객체 내부의 **모든 값**을 재귀적으로 비교
* JS 기본 제공 하지않음.

---

### 방법

#### 간단한 경우

```js
JSON.stringify(obj1) === JSON.stringify(obj2);
```

순서, 함수, undefined는 비교 불가

---

#### 라이브러리

```js
_.isEqual(obj1, obj2);
```

---

### 사용 예

* API 응답 비교
* 데이터 동등성 판단
